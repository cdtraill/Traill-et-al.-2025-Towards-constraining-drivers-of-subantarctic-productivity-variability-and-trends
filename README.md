# Traill-et-al.-2025-Towards-constraining-drivers-of-subantarctic-productivity-variability-and-trends
Contains Jupyter notebooks to generate all figures presented in Traill et al. (2025) "Towards constraining drivers of subantarctic productivity variability and trends".

Figures 1 and 2: Process OC-CCI NPP algorithms and contain links to relevant data repositories. 

Figures 3 - 7: Processes model output from mean monthly interannual hindcast (JRA-55) simulations in PISCES-Quota-Fe (Tagliabue et al., 2023). 

Figures 8 - 11: Process CMPI6 output from 8 earth system models.
